1. Open terminal

2. "cd ./src/HEMMS/server/"
3. "npm i express cors mongodb mongoose nodemon"
4. "npm i"
5. "npm run dev"

6. Split terminal

7. "cd ../Client/"

9. "ipconfig" and note IPv4 Address

10. Go to 'connection.js'
11. Replace line 2 with "export const url = "http://<Your_IPv4_Address>:" + port"

12. "npm start"
